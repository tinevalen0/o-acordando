# O Acordando

Um site simples e leve que inspira reflexões e bem-estar matinal.

## Tecnologias usadas
- HTML5
- CSS3

## Como usar
1. Baixe os arquivos clicando em **Code** > **Download ZIP**.
2. Extraia o ZIP no seu computador.
3. Abra o arquivo `index.html` no navegador.
